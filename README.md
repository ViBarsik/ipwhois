## Description
IpInfo repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## API
http(s)://{yourdomain}/ip/{:checked_ip}
- yourdomain - your local or production domain
- checked_ip - ip adress that you need get info

## EXAMPLE
http://localhost:3000/ip/8.8.8.4
